The MDB ESP32 Cashless project enables integration of cashless payment systems into vending machines using the ESP32 microcontroller.

Main features:
- MDB peripheral communication (Master/Slave)
- Wi-Fi and Bluetooth telemetry support
- Event and transaction logging
- Integration with external gateways
- Open-source under the MIT license

Ideal for developers and IoT integrators working with vending machines and commercial automation.
